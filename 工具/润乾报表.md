# 润乾报表

```
report-V2018.dmg仅应用于MAC环境，包含润乾报表V2018设计器、DQL元数据层设计器、DQL Server服务器、HSQL示例数据库、内置tomcat、DEMO应用、DEMO应用用到的HSQL示例数据库。
在MAC中除了可以通过应用程序->Report.app启动润乾报表设计器外，还可通过.sh文件启动其他程序，参考如下说明：    
    启动DQL元数据层设计器命令：sh /Applications/Report.app/Contents/raqsoft/report/bin/startup-dql.sh
    一键启动报表中心命令: sh /Applications/Report.app/Contents/raqsoft/report/bin/startup-reportcenter.sh，该命令会同时启动示例数据库和DQL Server服务
    启动示例数据库命令: sh /Applications/Report.app/Contents/raqsoft/report/bin/startDataBase.sh
    停止示例数据库命令: sh /Applications/Report.app/Contents/raqsoft/report/bin/stopDataBase.sh
    启动图形界面的DQL SERVER命令: sh /Applications/Report.app/Contents/raqsoft/report/bin/startDQLSERVER.sh
    启动非图形界面的DQL SERVER命令: sh /Applications/Report.app/Contents/raqsoft/report/bin/startDQLSERVER.sh -d
    启动润乾报表V2018的demo应用命令: sh /Applications/Report.app/Contents/raqsoft/report/bin/startup-demo.sh
    停止tomcat命令: sh /Applications/Report.app/Contents/raqsoft/report/bin/stop-demo.sh
    关闭DQL SERVER命令: sh /Applications/Report.app/Contents/raqsoft/report/bin/startDQLSERVER.sh -x
    启动润乾报表V2018的demo应用: sh /Applications/Report.app/Contents/raqsoft/report/bin/startup-demo.sh
    关闭报表中心: sh /Applications/Report.app/Contents/raqsoft/report/bin/stop-reportcenter.sh
    麒麟kylin linux+OpenJDK的环境启动润乾报表V2018设计器: sh /Applications/Report.app/Contents/raqsoft/report/bin/startup-report-kylin.sh
注意: 
    1. 在MAC中部署润乾报表V2018时需在当前环境中安装jdk，/Applications/Report.app/Contents/raqsoft/report/bin/目录下的config.txt文件中java_home路径默认修改为jdk路径，用户可手动修改java_home路径，例如：java_home=/Library/Java/JavaVirtualMachines/jdk1.7.0_80.jdk/Contents/；config.txt文件主要用来启动内置tomcat时脚本的引用。
    2. 如需将DEMO单独部署到独立的tomcat中，部署流程可参考在线文档【程序员参考】：http://doc.raqsoft.com.cn/report/preference/ 
    3. 查询和分析功能的web界面地址:      
      分组分析url：http://localhost:6868/demo/raqsoft/guide/jsp/olap.jsp
      明细查询url：http://localhost:6868/demo/raqsoft/guide/jsp/qyx.jsp

```
