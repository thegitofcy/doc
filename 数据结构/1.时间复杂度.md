# 00.【复杂度分析】
`O(1), O(n), O(lgn), O(nlogn), O(n^2)`

O : 大写英文字母 o, 描述的是算法的运行时间和输入数据之间的关系.描述的是渐进时间复杂度.

==**例如以下代码 :**==

```java
public static int sum(int[] nums){
    int sum = 0;
    for (int num : nums) {
        sum += num;
    }
    return sum;
}
```
n : 是 nums 数组中元素的个数. 以上算法运行的时间多少是和元素的个数称`线性关系`.  也就是说nums 中的元素越多, 那么算法执行的时间也就越长. 所以以上算法的复杂度为 `O(n)`.
